# Thuật toán sắp xếp nổi bọt 
* Cho dãy N số cần sắp xếp
* Lặp đi lặp lại N - 1 bước:
  * Với mỗi cặp kề nhau, nếu số bên trái lớn hơn số bên phải thì hoán đổi vị trí cho nhau.
 * Độ phức tạp thuật toán: $O(n^2)$
